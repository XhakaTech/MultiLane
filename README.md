# MultiLane

Multi-lane repository system for managing complex workflows.

## Project Structure

- **Assets/**: Game assets, audio files, sprites, and configurations
- **Docs/**: Documentation files including architecture and design documents
- **Packages/**: External packages and dependencies
- **ProjectSettings/**: Project configuration files
- **Tools/**: Utility scripts and tools

## Getting Started

This is a Unity project with additional tools for workflow management.
