# MultiLane Changelog

## [1.0.0] - 2026-05-28

### Initial Release

#### Added
- Initial project structure from Indoctrination
- Sphere character system with physics-based movement
- Thruster-based propulsion system with 4 directional controls
- Fuel consumption and regeneration system
- Voxel world destruction system
- Multi-material system with damage profiles
- Comprehensive documentation
- Localization tools (Spanish, French, German, Indonesian, Italian, Polish, Portuguese, Turkish, Vietnamese)

#### Documentation
- Complete architecture documentation for Sphere entity
- Voxel world design document
- Component references and data flow diagrams

#### Tools
- Localization synchronization script
- Editor tools for project setup
